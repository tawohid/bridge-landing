# Bridge Landing Page

> To be *heard* is to be *loved*.

A waitlist landing page for Bridge — facilitating the conversations that matter most.

## Deploy

Hosted on [Netlify](https://netlify.com) at [bringbridge.com](https://bringbridge.com).

## Waitlist

Email submissions are collected via [Netlify Forms](https://docs.netlify.com/forms/setup/). 
View submissions in the Netlify dashboard under **Forms → waitlist**.

To get email notifications: Netlify Dashboard → Site → Forms → Form notifications → Add email notification.
